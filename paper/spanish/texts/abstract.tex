\begin{abstract}

\textit{Propósito:} Crear arquetipos openEHR a partir de definiciones de recursos FHIR que permitan integrar repositorios que obedecen el modelo de referencia de openEHR a través de interfaces de FHIR.

\textit{Métodos:} Un prerrequisito para convertir instancias de recursos FHIR a instancias de arquetipos openEHR es encontrar equivalencias entre los tipos de datos FHIR y openEHR. Se propone las condiciones para establecer dichas equivalencias. Adicional, se presenta una forma de creación de arquetipos openEHR utilizando equivalencias de tipos.

\textit{Resultado:} Un proceso de creación automático basado en un sistema de tipos con soporte para vinculaciones a terminologías.

\textit{Conclusión:} Existe compatibilidad completa entre los tipos de datos de FHIR y openEHR, posibilitando la creación de arquetipos openEHR de recursos FHIR.

\end{abstract}
